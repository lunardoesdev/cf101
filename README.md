# cf101

To generate your clash-comptible WARP configuration, you can run
the following command if you have uv installed:
```
uvx git+https://github.com/lunardoesdev/cf101@main
```

Or, you can just install cf101 package:
```
pip install git+https://github.com/lunardoesdev/cf101@main
```
and then run `cf101`

After that, simply import that configuration by qr code or direct link using your favourite clash-compatible app.
